Projeto em MikroC.

O funcionamento parte do principio de ativação em duas vias, a principal e a de segurança, caso só a principal seja acionada, o sistema irá ativar os relês, mas por
alguns segundos, onde se dá o alerta, para ativação da segunda via, caso não seja acionada a tempo, o sistema automáticamente corta a energia dos relês.

Usando o princípio de esouro do contador do PIC.
